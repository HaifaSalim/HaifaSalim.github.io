Aircraft Management System

This is an information system for the management of aircraft during their arrival, unloading, loading and departure from an Airport. This project is a support system for the people associated with the airport and aircraft: it provides information for the staff, passengers and non-travelling public about the status of aircraft, and regulates the progress of the aircraft through the various stages of its visit.The system features real-time status monitoring and has role-specific interfaces, adhering to UML-based architectural specifications for aircraft lifecycle management.

Key challenges faced included, interpreting the complex UML design, implementing the MVC structure across multiple interfaces, and maintaining synchronized data updates. Managing aircraft state transitions, debugging interconnected components, and coordinating team work through GitHub under time constraints also proved demanding.

Key Features

  • State Management and Event Handling
  • Role-Based Interfaces
  • Version Control and Team Collaboration
  • Simulation of Real-World Systems
  • Testing and Validation (JUnit)
  • Centralized Aircraft Management Database
