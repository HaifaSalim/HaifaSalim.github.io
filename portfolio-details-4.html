2D Platform Game

“A Penguin Adventure” is a 2D platform game built using the GameCore library. The player controls a penguin navigating through two levels, collecting coins, avoiding obstacles, and defeating enemies (yeti sprites). The game combines animation, sound, and parallax-scrolling backgrounds to create an immersive experience.

The main challenges in developing A Penguin Adventure involved implementing accurate collision detection between sprites and tiles, synchronizing animations with player actions, and balancing sound effects with background music to maintain immersion. Additional difficulties included creating smooth parallax scrolling for visual depth, designing engaging levels with appropriate difficulty progression, and ensuring seamless interaction between multiple game elements such as enemies, coins, and player controls.
