AI-PERSONALIZED HOTEL BOOKING PLATFORM

The main aim of this project is to create a platform that will be able to offer personalised hotel options using artificial intelligence for a better user experience. The platform will be able to provide personalized recommendations with hotel suggestions, a booking system that has dynamic search functions, and AI user support through a chatbot.

One of the main challenges faced was the integrating the data for the hotels, rooms, and theuser data. While the hotel data was available, data regarding rooms had to be manually sourced from the hotel’s website. While the platform compares well to other hotel booking platforms in terms of functionality, there are still areas for improvement, such as the incorporation of real time data, payment integration, etc. Among the challenges that arise when personalizing recommendations is maintaining the platform's performance steadily and how scalable the platform is. The hotel recommendations should remain relevant to the users' needs and the platform should be secure and be compliant with GDPR and protect the user’s data

Key Features

  • Hotel Search and Booking
  • User Profiles and Personalization
  • Secure Authentication
  • AI Chatbot (Powered by Gemini)
  • AI-Based Recommendation System
  • Admin Dashboard
